# UEEA2601 上传功能模块 — 集成说明

## 完整流程

```
用户上传边界 → 点"开始分析"
  → 优先用用户自己的 GEE 密钥（方案A），没有则用平台公共账号（限频）
  → GEE 云端计算（建设用地 / RSEI / 夜灯 / 人口...）
  → 导出到 Google Cloud Storage → 后端下载
  → GeoServer 发布 WMS + 自动配色（SLD）
  → 生成分析报告（地图 + 图表 + 指标）
  → 前端拿到报告 JSON → 渲染专属页面
```

## 📦 集成步骤（3步）

### 第1步：复制 `app/` 到项目根目录

### 第2步：执行 SQL
```bash
docker compose up -d
psql -h localhost -U ueea2601 -d ueea2601 -f sql/add_user_boundaries.sql
```

### 第3步：修改 main.py

```python
# 1) 顶部 import
from app.database import init_db
from app.routers import upload, boundary, compute, gee_key, health

# 2) FastAPI app 之后
@app.on_event("startup")
def on_startup():
    init_db()

# 3) 路由区域
app.include_router(upload.router)
app.include_router(boundary.router)
app.include_router(compute.router)
app.include_router(gee_key.router)
app.include_router(health.router)
```
